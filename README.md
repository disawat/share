📌 Project Overview

This project analyzes housing prices across major U.S. cities to understand how population size influences property values.


🎯 Objectives

• Identify correlations between population and housing prices

• Visualize geographic price distribution

• Provide insights for buyers and sellers


🛠 Tools & Technologies

• Python (Pandas, Matplotlib, Seaborn)

• Tableau (for visualization)


📊 Key Insights

• Higher population cities tend to have higher housing prices

• Some mid-sized cities show strong value opportunities

• Geographic patterns reveal regional price clusters
